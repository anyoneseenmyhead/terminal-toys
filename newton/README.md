# Newton

Newton is a small Rust project that runs in your terminal. It renders an interactive newton experience in the console.

## Run

```sh
cargo run --release
```

## Controls

- Q: Quit
- Space: Toggle pause
- R: Reset
- C: Wrapping add
- A: Pluck
- D: Len
- + / =: Add ball
- -: Remove ball
- G: Is some
- Up: Increase rest
- Down: Decrease rest
- Left: Decrease air drag
- Right: Increase air drag
