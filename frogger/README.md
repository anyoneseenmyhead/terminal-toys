# Frogger

Frogger is a small Rust project that runs in your terminal. It renders an interactive frogger experience in the console.

## Run

```sh
cargo run --release
```

## Controls

- Q: Quit
- R: Reset run
- T: Wrapping add
- Space: Toggle pause
- W / Up: Move frog
- S / Down: Move frog
- A / Left: Move frog
- D / Right: Move frog
