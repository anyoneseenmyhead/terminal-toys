# Planetarium

Planetarium is a small Rust project that runs in your terminal. It renders an interactive planetarium experience in the console.

## Run

```sh
cargo run --release
```

## Controls

- Q: Quit
- C: Increase planet idx
- 1: Reset planet idx
- 2: Min
- 3: Min
- 4: Min
- 5: Min
- 6: Min
- 7: Min
- 8: Min
- 9: Min
- 0: Toggle alien mode
- Space: Toggle pause
- Left: Decrease rot speed
- Right: Increase rot speed
- Up: Increase tilt
- Down: Decrease tilt
- Ctrl+R: Map
