# Aurora

Aurora is a small Rust project that runs in your terminal. It renders an interactive aurora experience in the console.

## Run

```sh
cargo run --release
```

## Controls

- Q: Quit
- Space: Toggle pause
- H: Toggle help
- S: Toggle stars
- Up: Increase activity
- Down: Decrease activity
- Left: Decrease wind
- Right: Increase wind
- [: Decrease curtains
- ]: Increase curtains
- -: Decrease height
- =: Increase height
- ,: Decrease hue shift
- .: Increase hue shift
- R: Clear
