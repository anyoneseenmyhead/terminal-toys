# Fountain

Fountain is a small Rust project that runs in your terminal. It renders an interactive fountain experience in the console.

## Run

```sh
cargo run --release
```

## Controls

- Q / Esc    quit
- Space      pause
- C          cycle theme
- Up/Down    height up/down
- Left/Right width narrower/wider
- M          toggle mist
- R          reset (new seed)
- H          toggle this help
- Tip: resize the terminal to change the fountain scale
