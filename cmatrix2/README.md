# Cmatrix2

Cmatrix2 is a small Rust project that runs in your terminal. It renders an interactive cmatrix2 experience in the console.

![cmatrix2 screenshot](../screenshots/cmatrix2.png)

## Run

```sh
cargo run --release
```

## Controls

- Q / Esc: Quit
- C: Next
- + / =: Decrease frame ms
- -: Increase frame ms
- ]: Increase density
- [: Decrease density
- D: Toggle double enabled
- S: Toggle shimmer
- R: From entropy
