# Fluidlite Braille

Fluidlite Braille is a small Rust project that runs in your terminal. It renders an interactive fluidlite braille experience in the console.

![fluidlite-braille screenshot](../screenshots/fluidlite.png)

## Run

```sh
cargo run --release
```

## Controls

- Q: Quit
- P: Toggle pause
- D: Toggle debug
- C: Toggle vorticity
- F: Toggle continuous emit
- X: Next theme
- R: Reset
- Space: Inject
- Up: Decrease emitter y
- Down: Increase emitter y
- Left: Decrease emitter x
- Right: Increase emitter x
