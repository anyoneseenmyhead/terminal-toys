# Plasmaglobe

Plasmaglobe is a small Rust project that runs in your terminal. It renders an interactive plasmaglobe experience in the console.

## Run

```sh
cargo run --release
```

## Controls

- Q: Quit
- C: Increase palette i
- + / =: Increase streamers target
- - / _: Decrease streamers target
- H: Toggle HUD
- G: Toggle glass
- B: Toggle branches
- D: Toggle disco mode
- P: Toggle pause
- N: Enable step once
- [: Decrease sim speed
- ]: Increase sim speed
