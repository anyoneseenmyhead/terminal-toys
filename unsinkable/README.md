# Unsinkable

Unsinkable is a small Rust project that runs in your terminal. It renders an interactive unsinkable experience in the console.

![unsinkable screenshot](../screenshots/unsinkable.png)

## Run

```sh
cargo run --release
```

## Controls

- Q: Quit
- Space: Toggle pause
- R: Reset
- F: Flip
- C: Jostle
- Left: Enable left down
- Right: Enable right down
- Down: tilt_target *= 0.85
- Up: tilt_target *= 0.95
- 0: Set quality
- 1: Set quality
- 2: Set quality
