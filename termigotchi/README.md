# Termigotchi

Termigotchi is a Tamagotchi-like digital pet for the terminal. Feed, play, clean, and care for your pet as it grows.

![termigotchi screenshot](../screenshots/termigotchi.png)

## Run

```sh
cargo run --release
```

## Controls

- Q: Quit
- H: Toggle help
- F: Feed
- P: Play
- C: Clean
- M: Medicine
- S: Sleep toggle
- Tab: Settings

## Settings

- Up / Down: Select item
- Enter: Apply
- Tab / Esc: Back

## Rename

- Type to edit the name
- Enter: Save
- Esc: Cancel
- Backspace: Delete
