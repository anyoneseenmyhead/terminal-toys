# Grayscott

Grayscott is a small Rust project that runs in your terminal. It renders an interactive grayscott experience in the console.

![grayscott screenshot](../screenshots/grayscott.png)

## Run

```sh
cargo run --release
```

## Controls

- Q: Quit
- Space: Toggle pause
- R: Reset
- C: Increase contrast
- X: Decrease contrast
- [: Decrease v mid
- ]: Increase v mid
- -: Decrease steps per frame
- =: Increase steps per frame
- P: Reset
- T: Increase theme idx
- Up: Increase feed
- Down: Decrease feed
- Right: Increase kill
- Left: Decrease kill
- ,: Decrease paint radius cells
- .: Increase paint radius cells
