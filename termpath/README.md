# Termpath

Termpath is a small Rust project that runs in your terminal. It renders an interactive termpath experience in the console.

## Run

```sh
cargo run --release
```

## Controls

- Q: Quit
- Left: Move cursor
- Right: Move cursor
- Up: Move cursor
- Down: Move cursor
- Space: Toggle draw walls
- X: Idx
- S: Idx
- E: Idx
- R: Set status
- C: Len
- B: Start search
- D: Start search
- A: Start search
