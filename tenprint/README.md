# Tenprint

Tenprint is a small Rust project that runs in your terminal. It renders an interactive tenprint experience in the console.

![tenprint screenshot](../screenshots/tenprint.png)

## Run

```sh
cargo run --release
```

## Controls

- Q / Esc: Quit
- Space: Toggle pause
- H: Toggle help
- R: Reseed
- C: Reseed
- M: Reseed
- S: Reseed
- F: Increase fg idx
- B: Increase bg idx
- + / =: Decrease frame ms
- - / _: Increase frame ms
